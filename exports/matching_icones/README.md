# Matching icônes / boutons Land2Map

Dossier de suivi du matching entre les icônes disponibles et les boutons Land2Map à créer.

Source locale générée : `matching_icones_boutons_land2map.xlsx`.

Contenu du classeur source :

- `Synthese`
- `Mapping_objets`
- `Mapping_lineaires`
- `Sans_icone`
- `Icones_disponibles`
- `Icones_non_utilisees`
- `Regles_alias`
- `Mode_emploi`

Règle appliquée : aucune correspondance inventée. Lorsqu’aucune icône claire n’est trouvée, la valeur reste vide.

Résultat de synthèse :

- Objets palette : 90 matchs, 67 sans icône, 1 ambigu.
- Linéaires : 3 matchs directs, 34 sans icône spécifique.

Les exports CSV détaillés sont à pousser dans ce dossier avec le même séparateur `;`.
